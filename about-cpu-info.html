The cpu is the heart of a computer . It is responsible for the processing of instruction and data, and for the execution of program.

What is primary memory

Primary memory is the stores the data that is being processed by the CPU. It is also known as the "working memory".
primary memory, also known as main memory or RAM (Random Access Memory).

What is secondary memory?

Secondary memory is the memory that stores the instruction that the CPU executes. It is also known as the "program memory".
secondary memory , also known as secondary memory or ROM (Read only memory).
