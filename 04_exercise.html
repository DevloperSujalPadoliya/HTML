Key Computer Science Terms

Algorithm
A step-by-step procedure to solve a problem or perform a computation.
Database
An organized collection of structured information or data, typically stored electonically.
Compiler
A program that translates source code written in one programming language into another language.
APL
Application Programming Interface; a set of function and protocols for building software application.
